\section{Applications}

\subsection{Fractal Generation}
IFNs provide a natural framework for generating fractals with controlled local geometry. The graph structure allows for more flexible self-similarity patterns than classical IFS.

\begin{example}
Consider $X=\R^2$ with $G$ a 4-cycle and functions:
\[ f_k(x,y) = \frac{1}{3}(x\cos(\theta_k) - y\sin(\theta_k), x\sin(\theta_k) + y\cos(\theta_k)) \]
where $\theta_k = \frac{k\pi}{2}$. This IFN generates a novel fractal with rotational symmetry.
\end{example}

\subsection{Optimization}
IFNs can be used to design parallel fixed-point algorithms. The graph structure enables parallel updates while maintaining convergence guarantees.
